# Mathe 3D

Kleine C++ Header-only Library für 3D Berechnungen.

## Berechnungen
- Vektoraddition
- Vektorsubtraktion
- Multiplikation mit Skalar
- Division mit Skalar
- Vektorlänge
- Vektor normalisieren
- Punktprodukt/Skalarprodukt

## Befehle
### Kompilieren, Aufräumen und Ausführen
$ make
### Kompilieren
$ make compile
### Aufräumen
$ make clean
### Ausführen
$ make run
